## Codebase Patterns
- Path alias `@/*` maps to `./src/*` (configured in tsconfig.json)
- Components are in `src/components/`, lib utilities in `src/lib/`
- The `ralph/` subdirectory is a separate project (flowchart) — not part of the main app, and its TS errors can be ignored
- package-lock.json is in .gitignore — don't try to stage it
- Git user identity is not configured globally — must set per-repo before committing
- All components use Neo-Brutalism styling: thick borders (2-5px), box-shadow offsets, uppercase tracking-wide text
- `use client` directive is used on interactive components (wizard, modals, PWA prompt)
- Header and Footer are server components (no 'use client' directive)
